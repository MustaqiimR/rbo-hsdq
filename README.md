# rbo-hsdq
High-speed piezoelectric data acquisition front-end for ESP32, featuring a low-noise OPA320 amplifier, mid-rail biasing, anti-aliasing filters, and a 16-bit 1 MSPS SAR ADC (MCP33131D-10). Includes hardware design (KiCad), firmware examples (Arduino/ESP-IDF), and documentation for vibration, impact, and acoustic emission sensing.
